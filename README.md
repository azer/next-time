## next-time

Returns the nearest date of given time

## Install

```bash
$ npm install next-time
```

## Usage

At 6:00:

```js

now
// => 20:30

nextTime('23')
// => Date { Today 23:30 }

nextTime('7:20')
// => Date { Tomorrow 7:20 }
```
