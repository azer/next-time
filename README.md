## next-time

Returns the nearest date of given time

## Install

```bash
$ npm install next-time
```

## Usage

```js

now
// => 20:30

nextTime('23')
// => Date { Today 23:30 }

nextTime('11:09am')
// => Date { Today 11:09am }

nextTime('7:20pm')
// => Date { Tomorrow 7:20pm }
```
